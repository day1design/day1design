연결 가이드

  1. 도메인(day1design.co)을 Vercel 프로젝트에 연결합니다.
  2. 구글 애널리틱스(GA4) 속성을 생성하고 측정 ID를 받아 사이트 HTML 공통 헤더에 삽입합니다.
  3. 구글 서치 콘솔에서 동일 도메인을 등록하고 사이트맵을 제출합니다.
  4. 구글 클라우드 콘솔에서 서비스 계정을 생성하고, 애널리틱스 · 서치 콘솔의 읽기 권한을 부여합니다.
  5. 서비스 계정 키를 Worker 시크릿으로 등록하면 이 페이지 상단 유입통계가 실제 값으로 표시됩니다.
