연결 가이드

  1. 도메인(day1design.co)을 Vercel 프로젝트에 연결합니다.
  2. Google Analytics 4 속성을 생성하고 측정 ID를 받아 HTML에 삽입합니다.
  3. Search Console에서 동일 도메인을 등록하고 사이트맵을 제출합니다.
  4. Google Cloud Console에서 서비스 계정을 생성하고, GA4 · Search Console의 읽기 권한을 부여합니다.
  5. 서비스 계정 키를 Worker 시크릿에 저장하면 이 페이지에서 실제 값이 표시됩니다.
