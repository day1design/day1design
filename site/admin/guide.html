데이원디자인 관리자 사용 가이드입니다. 로그인부터 각 메뉴 사용법까지 순서대로 정리되어 있으니, 궁금한 항목을 찾아 보세요. 더 자세한 문의나 문제가 생기면 아래 운영방으로 알려 주시면 빠르게 도와드립니다.

1로그인 하기

처음 들어오거나 로그아웃한 상태면 로그인 화면이 먼저 나옵니다.

  1. 브라우저에서 admin-day1design.vercel.app으로 접속합니다. (즐겨찾기 해두면 편합니다.)
  2. 전화번호 칸에 관리자용 휴대폰 번호를 입력하고 인증 코드 받기 버튼을 누릅니다.
  3. 등록된 텔레그램 계정으로 6자리 숫자 코드가 도착합니다. (텔레그램 앱이 휴대폰에 있어야 받을 수 있습니다.)
  4. 받은 6자리 숫자를 입력하고 로그인을 누르면 들어갑니다.
참고. 한 번 로그인하면 같은 브라우저에서는 일정 시간 자동 유지됩니다. 공용 PC라면 사용 후 오른쪽 아래 로그아웃을 꼭 눌러 주세요.

2대시보드 (홈 화면)

로그인 직후 보이는 화면으로, 전체 현황을 한눈에 확인할 수 있습니다.

화면 구성

  • 유입 요약 — 방문자·페이지뷰 등 (GA4 연동 이후 자동으로 표시됩니다. 지금은 모두 “—”로 보입니다.)
  • 상담 접수 — 최근 30일 동안 들어온 총 접수 수, 홈페이지·Meta 광고 각각의 비중, 아직 응대하지 않은 접수대기 건수.
  • 최근 상담 접수 — 가장 최근 5건의 이름·출처·상태 미리보기. 전체 보기 →를 눌러 상담신청 메뉴로 이동합니다.
  • 관리 바로가기 — 히어로 슬라이드·포트폴리오· 커뮤니티·상담신청 메뉴로 빠르게 이동하는 카드.

3히어로 슬라이드

홈페이지 메인 상단에서 자동으로 넘어가는 큰 이미지 슬라이드를 관리합니다. 최대 10장까지 등록 가능합니다.

순서 바꾸기

  1. 슬라이드 카드를 마우스로 끌어다 놓으면 순서가 바뀝니다.
  2. 저장 버튼을 누르지 않아도 순서는 곧바로 반영됩니다.

이미지 교체 / 수정

  1. 카드의 수정(또는 카드 클릭)을 누릅니다.
  2. 팝업에서 이미지 업로드·설명 문구(alt)· 이동 링크(href)를 바꿀 수 있습니다.
  3. 저장을 누르면 홈페이지에 즉시 반영됩니다.

새 슬라이드 추가

  1. 마지막의 + 추가 버튼을 누릅니다.
  2. 이미지 파일을 선택하고 설명·링크를 입력 후 저장합니다.
이미지 팁. 가로로 긴 와이드 이미지(예: 가로 2400px 안팎)가 가장 잘 어울립니다. 파일 크기는 업로드할 때 자동으로 줄여주니 너무 걱정하지 않으셔도 됩니다.

4포트폴리오

시공 완료 사례를 등록·편집하는 메뉴입니다. 포트폴리오 페이지의 카드 순서도 이곳에서 정합니다.

새 프로젝트 등록

  1. 상단 + 새 프로젝트를 누릅니다.
  2. 카테고리(HOUSE / OFFICE), 프로젝트 이름, 평수를 선택합니다.
  3. 대표 이미지와 상세 이미지를 업로드합니다. 여러 장을 한 번에 선택해도 됩니다.
  4. 저장을 눌러 마무리.

순서 바꾸기

카드를 마우스로 끌어 놓으면 사이트에 보이는 순서가 바뀝니다.

기존 프로젝트 수정 / 삭제

카드를 클릭해서 편집 화면으로 들어간 뒤, 내용을 고치고 저장하거나 아래의 삭제 버튼을 눌러 제거할 수 있습니다.

5커뮤니티

블로그처럼 글을 쓰는 공간입니다. 공간 제안·시공 후기·인사말 등을 게시합니다.

새 글 쓰기

  1. 상단 + 새 글 작성을 누릅니다.
  2. 제목·카테고리·썸네일을 먼저 채워주세요. 썸네일은 목록에서 보이는 작은 이미지입니다.
  3. 본문은 블록 단위로 쌓습니다. + 버튼을 눌러 원하는 블록을 추가하세요.
    • 텍스트 — 문장/문단
    • 이미지 — 한 장은 큰 이미지, 여러 장을 올리면 자동으로 갤러리(2~4칸)로 배치됩니다.
  4. 오른쪽 위 저장(공개)·임시저장(나만 보기) 중 선택합니다.

글 수정 / 삭제

목록에서 글을 클릭하면 수정 화면이 열립니다. 내용을 바꾼 뒤 저장하거나, 하단의 삭제 버튼으로 제거할 수 있습니다.

6상담신청 (접수 관리)

홈페이지 견적문의 폼과 Meta 광고로 들어온 상담 접수를 한 곳에서 관리합니다. 가장 많이 쓰는 메뉴입니다.

목록 화면

  • 상단 필터로 상태(접수대기·상담중·견적완료· 계약완료·취소)와 출처(홈페이지/Meta) 를 좁혀볼 수 있습니다.
  • 검색 칸에 이름·연락처·주소·캠페인명을 입력하면 해당 항목이 포함된 건만 보입니다.
  • 행을 클릭하면 오른쪽에 상세 정보가 열립니다.

고객 정보 수정

상담 중 실제로 확인한 이름·연락처·주소·평수 등이 접수 내용과 다를 때 쓰세요. 상세 패널의 고객 정보 수정 칸에서 값을 고친 뒤 고객 정보 저장을 누르면 됩니다.

상담 관리 (상태 · 담당자 · 금액)

  • 상태 — 진행 단계를 바꿉니다. 최근 접수에서 “접수대기”가 오래 남아 있지 않게 관리하세요.
  • 담당자 — 누가 응대 중인지 기록.
  • 첫 연락 일시 — 고객과 처음 통화/회신한 시각. 응답 속도 파악에 사용합니다.
  • 견적 금액 — 견적이 확정되면 입력합니다. 대시보드 통계와 연결됩니다.

입력이 끝났으면 상담 정보 저장을 누르세요.

메모 (상담 기록)

  • 통화 내용·요청 사항 등을 짧게 남겨 두는 공간입니다. 여러 명이 보는 기록이므로 민감한 내용은 피하고, 가급적 간결하게 작성해 주세요.
  • 작성자 이름을 함께 남기면 나중에 누가 남긴 메모인지 구분하기 좋습니다. Ctrl+Enter 로도 저장됩니다.

회차 조회 (같은 고객 이력)

같은 연락처로 이전에 문의한 내역이 있으면 상세 상단에 자동으로 표시됩니다. 과거 상담 내용을 빠르게 확인할 수 있습니다.

고객 삭제

접수 정보를 영구 삭제할 때 사용합니다. 오른쪽 아래 빨간 고객 삭제 버튼을 누르면 확인 창이 한 번 더 뜨고, 확인 시 목록에서 즉시 사라집니다.

주의. 삭제하면 되돌릴 수 없고 메모·상담 이력도 함께 사라집니다. 테스트 접수나 명백한 스팸일 때만 사용하세요. 상담이 끝난 정상 고객은 상태를 계약완료 또는 취소로 바꾸는 편을 권장합니다.

7유입통계

사이트 방문 현황과 실제 접수 통계를 기간별로 확인합니다.

기간 선택

  • 7일 / 30일 / 전월 / 당월 / 전체 중 하나를 누르면 아래 통계가 그 기간으로 바뀝니다.
  • 선택기간을 누르면 시작·종료 날짜를 직접 골라 볼 수 있습니다.

유입 요약 (상단)

방문자·페이지뷰·평균 체류·이탈률·인기 페이지·유입 경로를 보여주는 영역입니다. 현재는 GA4(구글 애널리틱스) 연동 대기 중이어서 값이 비어 있습니다. 연동이 끝나면 자동으로 채워집니다.

상담 접수 통계 (하단)

  • 총 접수 / 홈페이지 / Meta 광고 / Meta 비중 4가지 숫자를 제일 먼저 보여줍니다.
  • 일별 접수 추이 그래프로 광고 반응과 자연 유입을 구분해서 볼 수 있습니다.
  • 상태별 분포 도넛 — 접수대기·상담중·견적완료 등 진행 단계 비율.
  • Meta 캠페인 TOP 5 — 어떤 광고 캠페인에서 가장 많은 상담이 들어왔는지 보여줍니다.

8공통으로 알아 두면 좋은 점

이미지 업로드 규칙

  • PC의 파일 선택으로만 올립니다. (외부 URL 입력 방식은 사용하지 않습니다.)
  • 업로드 순간 자동으로 용량이 줄고 최신 형식으로 바뀌어 올라가니, 원본 크기가 커도 괜찮습니다.
  • 교체·삭제한 이미지는 뒤에서 자동 정리됩니다. 따로 지워주지 않으셔도 됩니다.

저장 후 반영 시점

저장하면 홈페이지에는 길어야 1분 안에 반영됩니다. 바로 보이지 않으면 브라우저에서 새로고침(F5)을 한 번 해 주세요.

모바일 사용

스마트폰에서도 모든 기능을 쓸 수 있습니다. 좌측 상단 메뉴 아이콘을 눌러 다른 화면으로 이동하세요.

로그아웃

왼쪽 사이드바 아래 로그아웃 버튼을 누르면 세션이 종료됩니다. 공용 PC 사용 후에는 꼭 눌러 주세요.
