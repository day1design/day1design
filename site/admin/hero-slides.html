API 연동 가이드

관리자 UI는 아래 API 엔드포인트를 호출합니다. 서버 구현 시 이 스펙에 맞춰 응답해주세요. 엔드포인트가 없으면 자동으로 JSON 파일 fallback 모드로 동작합니다.

메서드 경로 역할
GET /api/hero/slides 현재 슬라이드 목록 조회 (hero-slides.json 구조와 동일)
PUT /api/hero/slides 전체 슬라이드 배열 저장 (body = hero-slides.json 구조)
POST /api/hero/upload multipart/form-data, field file. 응답 { url } 에 R2 public URL

인증은 서버에서 쿠키/Bearer 등 자체 결정. 클라이언트는 credentials: "include" 로 호출합니다.
